How to set a different page as your homepage?

1. Open the Pages panel (press "P" on the keyboard)
2. Open the settings of the page you want to make your new homepage
3. Set the Parent Folder to "None" and click to Save button
4. Then open the page settings again and click "Make 'This Page' Your Homepage"

How to edit side menu in Nav Bar v.2?

1. Double-click the Nav Bar v.2 symbol to open it
2. Switch to the Navigator panel (press "Z" on the keyboard) and select Nav Menu block
3. Go to the Style panel and set transform setting to "Hide"
4. Don't forget to change it back to "Visible" when you done

How to edit Subscribe Popup?

1. Open the Navigator panel (press "Z" on the keyboard)
2. Double-click "Subscribe Popup" symbol
3. Go to the Style panel and set it's layout setting to "Flex"
---

How to add a new Post Popup to the page

1. Open the Symbols panel (press "Shift + A" on the keyboard)
2. Drag and drop or just click to the Post Popup symbol to add it to your page
3. Go to the Interactions panel (press "H" on the keyboard)
4. Add a new Page Load trigger called "Post Popup Show" when page starts loading
+ We also recommend you to hide this interaction for mobile devices
